# Automation Exercise Cypress

Cypress automation framework for testing the Automation Exercise website.

## Technologies Used

- Cypress
- JavaScript

## Test Coverage

- Login functionality
- Signup functionality
- Product search
- Cart functionality
- API testing
- Intercept handling
- Assertions
- UI validations

## Installation

Install dependencies:

```bash
npm install
```

Run tests:

```bash
npx cypress open
```

Run in headless mode:

```bash
npx cypress run
```
